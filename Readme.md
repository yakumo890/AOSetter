# AOSetter

## 概要
あるオブジェクトの子のメッシュのアンカーオーバーライドを一括で設定するツール。

## 要求
- OS: windows
- powershell
- Unity: 2019.4.31.f
- VRChat Creator Companion

## プロジェクトの使い方
このプロジェクトはVRChat Creator Companion(VCC)により生成されている。
そのため、VCCに読み込ませて使用する。

### テスト
Test Runnerで単体テストが実行できる。

"Assets/Prefab"以下にテスト用のアバターのPrefab Variantが存在するので、これを用いて結合テストを行う(このPrefabはVRCSDKのサンプルアバターを参照している)。
"Yakumo890 > AOSetter"からツールを起動できる。

### Unitypackageのエクスポート
MakeUityPackage.ps1を実行すると、カレントディレクトリにUnitypackageが生成される。
